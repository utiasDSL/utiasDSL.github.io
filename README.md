# utiasDSL github.io forwarding

We switched from the new utiasDSL organization to the learnsyslab organization, and we want to make sure that all links to the old utiasDSL.github.io/xxx/ still work. This repository contains single forwarding files for all repositories that were moved, so that all links to the old utiasDSL.github.io/xxx/ will still work and redirect to the new learnsyslab.github.io/xxx/.

See https://inbo.github.io/.
